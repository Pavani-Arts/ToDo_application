# Smart Notes Todo App

A simple and smart todo application built with React and TypeScript.

## Features

- Add tasks
- Delete tasks
- Set priority levels: Normal, Important, Highly Important
- Clean and decent UI

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Build

To build the project for production:
```
npm run build
```

## Preview

To preview the production build:
```
npm run preview
```
